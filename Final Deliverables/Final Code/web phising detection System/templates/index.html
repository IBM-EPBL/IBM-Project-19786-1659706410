About Web Phishing Detection System

Web Phishing Detection System is a project that came into existence from IBM project “Detection of Phishing Website Using Machine Learning” carried out at MNMJEC. It aims to help reduce phishing attack by helping internet users authenticate URL link by testing if it is phishing or legitimate. The progress of validating a Website URL for phishing or legitimate has gone through several Machine learning models.

About the Author

The Author of this Project is a group of students who are a Final Year Students of Computer Science and Engineering at MNM Jain Engineering College who aspire to further his Career into Cyber Security.
